# Thorofare validation

Pin the exact [Thorofare](https://github.com/rpcpool/thorofare) commit and run it as a separate adjacent attempt on the same disclosed measurement host. Use processed RPCEdge/Triton endpoints, `--with-transactions`, and the Pump AMM filter where the pinned version supports it. Record the command with environment-variable placeholders, sample count, start time, commit and output links.

Thorofare’s slot or processed-transaction result can corroborate that scope. If the pinned version cannot express `SubscribeDeshred` with the same filter, deshred validation is `inconclusive`; do not emulate or relabel another stream.
